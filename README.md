# CFIR 2.0 MSL Situation Analysis Tool

基于 CFIR 2.0 框架的医学事务情境分析工具，帮助 MSL 和医学部团队系统化评估创新方案在目标医院的实施障碍与促进因素。

## 点击下载使用

👉 **[点击直接使用工具](https://lillianzww.github.io/cfir-msl-tool/)**

## 功能特点

- **CFIR 2.0 五维评估**：创新领域、外部环境、内部环境、个人领域、实施过程（完整 48 个构念）
- **ERIC 策略匹配**：基于评估结果自动推荐实施策略
- **AI 提示词生成**：一键复制结构化提示词，可粘贴到 ChatGPT/DeepSeek/Kimi 等大模型获取深度分析
- **雷达图可视化**：五维态势一目了然，快速识别短板与优势
- **纯离线运行**：所有数据仅存于本地浏览器，不上传任何服务器
- **中英双语支持**
- **手机端友好**：拜访路上也能用

## 工具亮点

- 每个构念配有医学事务语境下的中文定义与 MSL 实战举例
- 个人领域按"角色子域 + 特征子域（COM-B 模型）"结构化呈现
- 内部环境区分"持久特征（A-D）"与"实施相关（E-K）"
- 文化构念拆分为平等/患者/医护/学习四个子维度独立评分
- 可用资源拆分为资金/空间/材料与设备三个子维度独立评分
- 评估报告可导出为文本文件
